accept

@JvmOverloads fun accept(address: SocketAddress, packetProcessor: PacketProcessor = SingleThreadPacketProcessor(), pool: Pool<ReceivablePacket> = ByteBufferReceivablePacketPool()): Server

Platform and version requirements: JVM

Creates a new GServer with the passed arguments. If another implementation of Server is made default besides GServer, this function will change to reflect that.

Parameters

address - The SocketAddress to have the server accept on on enable.

packetProcessor - The PacketProcessor to use, defaulting to SingleThreadPacketProcessor.

pool - The Pool to get empty packets from.

Return
A Server that, when enabled, starts accepting clients.
