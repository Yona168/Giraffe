handle

abstract fun handle(client: Client, opcode: Opcode, packet: ReceivablePacket): Job

Platform and version requirements: JVM

Processes a ReceivablePacket by invoking a function, as registered with the passed Opcode, with the ReceivablePacket and an Client. This launches a new coroutine with coroutineContext context, and returns its Job.

Parameters

opcode - the Opcode to get the registered function with.

packet - The ReceivablePacket to handle.

client - the Client to pass into the function.

Return
Job of this coroutine
