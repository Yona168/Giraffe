QueuedOpSendablePacket

class QueuedOpSendablePacket : SendablePacket

Platform and version requirements: JVM

An implementation of SendablePacket that queues writing operations until building. This allows for multiple builds of a packet as well as easy ability to prepend information. This implementation builds a packet as follows:

  1. The opcode of the packet (of type Opcode).
  2. The size of the packet (of type Size).
  3. The rest of the bytes.
