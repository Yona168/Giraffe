newServerside

@JvmStatic @JvmOverloads fun newServerside(socketChannel: AsynchronousSocketChannel, packetProcessor: PacketProcessor, sessionUUID: UUID, pool: Pool<ReceivablePacket> = ByteBufferReceivablePacketPool()): GClient

Platform and version requirements: JVM

Creates a GClient from a given AsynchronousSocketChannel that will not connect on enable. It also has a given sessionUUID.

Parameters

socketChannel - The already-connected AsynchronousSocketChannel to use.

packetProcessor - The PacketProcessor to use.

sessionUUID - The UUID to set the client's Client.sessionUUID to

pool - The Pool to get empty packets from, defaulting to ByteBufferReceivablePacketPool

Return
A GClient to wrap the given channel.
