Messenger(packetProcessor: PacketProcessor, bufferPool: Pool<ReceivablePacket> = ByteBufferReceivablePacketPool())

Platform and version requirements: JVM

Provides base Component functionality for objects with AsynchronousChannels and a PacketProcessor. Enable/Disable functionality is first handled by ScopedComponent. On enable, packetProcessor is enabled. On disable, initClose is ran, and then socketChannel is closed

Parameters

packetProcessor - the PacketProcessor of this object

bufferPool - the Pool of ReceivablePackets to use to process packets with.
