Messenger
(JVM)

 abstract class Messenger : ScopedComponent, CanProcessPackets

Provides base Component functionality for objects with AsynchronousChannels and a PacketProcessor. Enable/Disable functionality is first handled by ScopedComponent. On enable, packetProcessor is enabled. On disable, initClose is ran, and then socketChannel is closed

ScopedComponent
(JVM)

 abstract class ScopedComponent : Component, CoroutineScope

A CoroutineScope that gets a new job when it enables, and cancels it when it disables.
